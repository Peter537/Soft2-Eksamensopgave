# Requirements Documentation

Denne mappe indeholder kravdokumentation for MToGo.

## Dokumenter

## Kravtyper

### Funktionelle Krav

User stories beskriver hvad systemet skal gøre fra brugerens perspektiv.

### Tekniske Krav

Technical stories beskriver tekniske implementeringsdetaljer.

### Ikke-Funktionelle Krav

NFR'er beskriver kvalitetsattributter såsom Performance, Availability, Scalability, Security, Maintainability og Usability.
