# Key Performance Indicators (KPIs)

Dette dokument beskriver de vigtigste Key Performance Indicators (KPIs) for MToGo-platformen.

## 4 udvalgte KPI'er

Da vi ikke har for meget tid, har vi udvalgt 4 centrale KPI'er til fokus. Disse er valgt for at give et hurtigt overblik over platformens ydeevne.

| KPI                        | Description                                                                             | Data source |
| -------------------------- | --------------------------------------------------------------------------------------- | ----------- |
| Active customers (monthly) | Antal unikke customers, der har gennemført mindst én order inden for de sidste 30 dage. | Orders      |
| Orders per hour / day      | Antal orders i en periode, som viser efterspørgsel og peak-tidspunkter.                 | Orders      |
| Active partners            | Antal partners, der aktuelt er aktiveret til at modtage orders.                         | Partner     |
| Active agents              | Antal agents, der aktuelt er tilgængelige til at levere orders.                         | Agents      |

## KPI Oversigt

| KPI                                  | Description                                                                             | Data source  |
| ------------------------------------ | --------------------------------------------------------------------------------------- | ------------ |
| Gross merchandise value              | Samlet værdi af alle succesfulde betalte orders i en given periode.                     | Orders       |
| Active customers (monthly)           | Antal unikke customers, der har gennemført mindst én order inden for de sidste 30 dage. | Orders       |
| Orders per active customer (monthly) | Gennemsnitligt antal afsluttede orders pr. aktiv customer i en måned.                   | Orders       |
| Orders per hour / day                | Antal orders i en periode, som viser efterspørgsel og peak-tidspunkter.                 | Orders       |
| Open orders                          | Antal orders, der aktuelt er i gang (placed, accepted, prepared, picked up).            | Orders       |
| Rejection rate                       | Procentdel af placed orders, der bliver rejected.                                       | Orders       |
| Avg rating                           | Gennemsnitlig order-rating givet af customers.                                          | Feedback Hub |
| Review coverage                      | Procentdel af delivered orders, der har modtaget et review.                             | Feedback Hub |
| Active partners                      | Antal partners, der aktuelt er aktiveret til at modtage orders.                         | Partner      |
| Orders per partner                   | Gennemsnitligt antal orders, som hver aktiv partner håndterer.                          | Orders       |
| Partner revenue                      | Indtjening som partners får via platformen i en given periode.                          | Orders       |
| Active agents                        | Antal agents, der aktuelt er tilgængelige til at levere orders.                         | Agents       |
| Orders per agent per hour            | Gennemsnitligt antal delivered orders pr. arbejdstime pr. agent.                        | Agents       |
| Avg agent rating                     | Gennemsnitlig rating, som customers giver agents.                                       | Feedback Hub |
| Avg agent rating pr. agent           | Gennemsnitlig rating, som customers giver en agent.                                     | Feedback Hub |
| Avg bonus payout per agent           | Gennemsnitligt månedligt bonusbeløb pr. berettiget agent.                               | AgentBonus   |
